# Working with Go Modules

## Create: `go mod init`

```
> go mod init aModuleThing
```

## Fetch: `go get`

## List packages: `go list` (see `-m` opt)

## Build: `go build` 

## Build, and install the binary in GOPATH bin (and cache stuff): `go install`

## Execute without creating a binary: `go run` 

## Clean: `go tidy`
